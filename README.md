# Azure Ubuntu Docker Nginx Lab

## Overview
This project deploys an Ubuntu virtual machine in Microsoft Azure using Terraform. The VM is configured to run Docker and host an Nginx web server.

## Architecture
- Azure Resource Group
- Virtual Network
- Subnet
- Network Security Group
- Ubuntu VM
- Docker
- Nginx

- Ubuntu Linux
- Docker
- Nginx
- GitHub
